1) Visual Studio Project AadharMedNet.sln with an publish settings to deploy on Azure

2) Aadhar Number is a UID in India with all personal and biometric details of an Indian citizen.

2) It calls the uidai govt. web api to authenticate the Aadhar Number through OTP authentication. An XML based authentication  is there to test with dummy Aadhar Number. KLAuth works for real Aadhar Number 


3) Targets: .Net framework 4.5, Visual Studio 2012 or above

4) To start: 1) Download the AadharMedNet folder; 2) open in Visual Studio; 3) Deploy on IIS or run directly through Visual Studio on any browser.
